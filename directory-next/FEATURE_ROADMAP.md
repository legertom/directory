# Feature Roadmap for 1pg.directory

## üéØ Approved Features (Post-Launch)

### Phase 1: Quick Wins (First Month)

#### 1. ‚úÖ CSV Template Download
- **Free & Paid**: Download example CSV template
- **Implementation**: Generate sample CSV, add download button
- **Effort**: ~1 hour
- **Value**: Reduces user errors, improves onboarding

#### 2. ‚úÖ Entry Counter/Validation
- **Free & Paid**: Show "X of 204 entries" as user uploads, highlight issues
- **Implementation**: Parse CSV client-side, show count and validation
- **Effort**: ~2 hours
- **Value**: Prevent errors before generation

#### 3. ‚úÖ Custom Title (Paid Feature)
- **Free**: Fixed "DIRECTORY" title
- **Paid**: Custom title (e.g., "Company Directory", "Emergency Contacts")
- **Implementation**: Add text input in UI, pass to API
- **Effort**: ~3 hours
- **Value**: Personalization, clear paid value

#### 4. ‚úÖ Custom Colors (Paid Feature)
- **Free**: Default blue title color (#1F4685)
- **Paid**: Choose title color, accent colors
- **Implementation**: Color picker in UI, pass hex to PDF generation
- **Effort**: ~4 hours
- **Value**: Brand matching, visual customization

#### 5. ‚úÖ Multiple Page Support (Paid Feature)
- **Free**: Single page only (204 entries max)
- **Paid**: Auto-generate multiple pages for larger directories
- **Implementation**: Calculate pages needed, add page numbers
- **Effort**: ~6-8 hours
- **Value**: Handle enterprise-sized directories

### Phase 2: Medium Priority (Months 2-3)

#### 9. ‚úÖ Batch Processing (Paid Feature)
- **Paid Only**: Upload multiple CSV files, generate multiple PDFs, download as zip
- **Implementation**: Queue system, zip download
- **Effort**: ~8-10 hours
- **Value**: Time-saving for bulk operations

#### 11. ‚úÖ Google Sheets Integration (Paid Feature)
- **Paid Only**: Import directly from Google Sheets (no CSV export needed)
- **Implementation**: Google Sheets API, OAuth flow
- **Effort**: ~10-12 hours
- **Value**: Modern workflow integration, removes friction

### Phase 3: Advanced Features (Months 4-6)

#### 10. ‚úÖ API Access (Paid/Enterprise Feature)
- **Paid/Enterprise**: REST API for programmatic PDF generation
- **Implementation**: API key system, rate limiting, documentation
- **Effort**: ~12-16 hours
- **Priority**: Low (but valuable for enterprise)
- **Value**: Automation, integrations, enterprise sales

#### 12. ‚úÖ Save & Edit Templates (Paid Feature)
- **Paid Only**: Save CSV templates, edit and regenerate
- **Implementation**: Database (Vercel Postgres or Supabase), user accounts
- **Effort**: ~16-20 hours (includes DB setup)
- **Value**: Reusability, convenience
- **Note**: Requires database, heavier lift
